# tf.app.run

Runs the program with an optional 'main' function and 'argv' list.

### Aliases:

* `tf.app.run`
* `tf.compat.v1.app.run`
* `tf.compat.v2.compat.v1.app.run`

``` python
tf.app.run(
    main=None,
    argv=None
)
```
