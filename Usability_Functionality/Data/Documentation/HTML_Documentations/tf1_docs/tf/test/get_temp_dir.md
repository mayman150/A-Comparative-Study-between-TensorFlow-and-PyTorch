# tf.test.get_temp_dir

Returns a temporary directory for use during tests.

### Aliases:

* `tf.compat.v1.test.get_temp_dir`
* `tf.compat.v2.compat.v1.test.get_temp_dir`
* `tf.test.get_temp_dir`

``` python
tf.test.get_temp_dir()
```
