# tf.contrib.layers.summarize_collection

Summarize a graph collection of tensors, possibly filtered by name.

``` python
tf.contrib.layers.summarize_collection(
    collection,
    name_filter=None,
    summarizer=tf.contrib.layers.summarize_tensor
)
```
