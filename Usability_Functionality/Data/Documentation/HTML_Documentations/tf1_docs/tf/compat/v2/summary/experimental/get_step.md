# tf.compat.v2.summary.experimental.get_step

Returns the default summary step for the current thread.

``` python
tf.compat.v2.summary.experimental.get_step()
```

<!-- Placeholder for "Used in" -->


#### Returns:

The step set by `tf.summary.experimental.set_step()` if one has been set,
otherwise None.
