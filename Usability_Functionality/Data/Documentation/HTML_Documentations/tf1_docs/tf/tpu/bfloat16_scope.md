# tf.tpu.bfloat16_scope

Scope class for bfloat16 variables so that the model uses custom getter.

### Aliases:

* `tf.compat.v1.tpu.bfloat16_scope`
* `tf.compat.v2.compat.v1.tpu.bfloat16_scope`
* `tf.contrib.tpu.bfloat16_scope`
* `tf.tpu.bfloat16_scope`

``` python
tf.tpu.bfloat16_scope()
```
