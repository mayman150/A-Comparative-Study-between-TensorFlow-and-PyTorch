# tf.logging.log_if

Log 'msg % args' at level 'level' only if condition is fulfilled.

### Aliases:

* `tf.compat.v1.logging.log_if`
* `tf.compat.v2.compat.v1.logging.log_if`
* `tf.logging.log_if`

``` python
tf.logging.log_if(
    level,
    msg,
    condition,
    *args
)
```
